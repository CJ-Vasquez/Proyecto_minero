# 01 - INFORME DEL PROYECTO

## Documentos incluidos

### 1. `INFORME_PROYECTO_SISTEMA_LOGISTICO_MINERO.docx` [PRINCIPAL]

Informe tecnico completo del proyecto en formato editable, siguiendo la estructura solicitada por el curso:

1. Caratula
2. Tabla de Contenido
3. Lista de Figuras y Tablas
4. Resumen
5. Introduccion
6. Diagnostico (Analisis SEPTE del sector minero)
7. Objetivos (SMART)
8. Justificacion del Proyecto
9. Definicion y Alcance
10. Planeamiento y Control del Proyecto (Gantt + Matriz de Riesgos)
11. Producto (Arquitectura, 5 diagramas UML embebidos, codigo)
12. Recursos y Tecnologias (31 tecnologias)
13. Pruebas Funcionales (22 automatizadas + 25 funcionales)
14. Conclusiones
15. Recomendaciones
16. Bibliografia (18 referencias)
17. Anexos (A-H, incluye compendio de diagramas)

**Formato:** Arial 11pt, interlineado simple, margenes A4 (superior/inferior 3cm, derecho/izquierdo 2.5cm) conforme al Plan de Proyecto del curso.
